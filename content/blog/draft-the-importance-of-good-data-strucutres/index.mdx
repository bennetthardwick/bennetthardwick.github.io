We write to from them, we search through them, we mutate them, all for the sake of better reflec

```javascript
const data = [
    { key: 'one', value: 1 }
    { key: 'two', value: 2 }
    { key: 'three', value: 3 }
    { key: 'four', value: 4 }
    { key: 'five', value: 5 }
];

const key = 'three';
const value = data.find(x => x.key === key); // 3
```
